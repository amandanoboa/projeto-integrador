PROJETO INTEGRADOR - INTEGRAÇÃO ENTRE SISTEMAS E SINCRONIZAÇÃO DE DADOS

Autora: Amanda Noboa de Camargo

Descrição:
Este projeto tem como objetivo demonstrar a integração de dados entre um arquivo CSV e um banco de dados SQLite utilizando a linguagem Python.

Arquivos do projeto:

* integracao.py
* clientes.csv

Como executar:

1. Certifique-se de possuir Python instalado no computador.
2. Abra o terminal na pasta do projeto.
3. Execute o comando:

python integracao.py

Resultado esperado:

A aplicação realizará a leitura dos dados presentes no arquivo clientes.csv, criará automaticamente o banco de dados clientes.db e exibirá no terminal a mensagem de sucesso juntamente com os registros importados.
